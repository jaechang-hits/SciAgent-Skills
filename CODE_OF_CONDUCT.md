# Code of Conduct

## Our Standards

We are committed to a welcoming, inclusive community. Expected behavior:

- Be respectful and constructive in discussions
- Accept feedback gracefully
- Focus on what is best for the project and its users
- Show empathy toward other contributors

Unacceptable behavior includes harassment, personal attacks, and discriminatory language.

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, and discussions.

## Enforcement

Instances of unacceptable behavior may be reported by opening an issue or contacting the maintainers directly via GitHub. Maintainers will review and respond to all reports.
